# 客服数据日报看板 - 多项修复

## 问题1：日期选择器禁用状态不直观
范围外的日期虽然不可点击，但看不出置灰效果，用户不知道哪些日期是禁用的。

## 问题2：手机端真实体验分指标名称不显示
手机端浏览时，真实体验分的卡片只显示数值，看不到指标名称（如"宝贝质量"、"物流速度"等）。

## 问题3：缺少当日总结和建议模块
页面底部需要加一个"当日总结和建议"区域，字数50字左右，根据当日数据自动生成一句话总结。

## 修复要求
1. 日期选择器弹窗中，禁用日期要有明显的灰色/半透明置灰样式，与可用日期区分开
2. 手机端真实体验分卡片中，指标名称和数值都要显示（可缩小字体，但必须有）
3. 页面最底部新增"当日小结"模块，包含当日总结和建议，字数50字左右，根据当日数据内容生成
