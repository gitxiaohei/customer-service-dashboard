# 客服数据日报看板 - UI 升级需求

## 用户反馈
1. 界面升级成市面上最炫酷的风格，参考高端数据大屏/现代BI看板
2. 手机打开也要能一眼看到核心指标（移动端优先设计）
3. 重要异常数据突出显示

## 设计参考方向
- 风格：深色主题 + 渐变卡片 + 玻璃拟态（glassmorphism）或新拟态（neumorophism）
- 配色：深色背景 + 霓虹渐变点缀（蓝/紫/青），或高级莫兰迪深色系
- 字体：大数字 + 粗体标题，数据层次分明
- 图表：渐变填充图表，动态光晕效果
- 布局：顶部核心指标横幅 → 中间图表 → 底部明细，竖屏手机优先

## 核心指标（手机上第一屏就要看到）
- 整体满意率：96.47%（异常⚠️）
- AI智能组满意率：88.00%（异常⚠️）
- 当日转化率：41.34%（异常⚠️）
- 月均转化率：50.03%
- 真实体验分综合评级

## 页面结构
1. 顶部：日期选择 + 核心指标大字横幅（3-4个最关键的）
2. 中部：满意率横向条形图 + 转化率柱状图
3. 底部：真实体验分明细指标网格
4. 全局：玻璃拟态卡片，圆角，阴影，渐变
